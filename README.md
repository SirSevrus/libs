# libs

Welcome to the `libs` repository! This repository contains a collection of sub-library Python files designed to perform short tasks and save you time. Whether you're looking for utility functions, helper classes, or specialized tools, you might find what you need here.

## Installation

To use any of the libraries in this repository, simply clone or download the repository to your local machine:

```bash
git clone https://github.com/SirSevrus/libs.git
```
## importing the libraries

To import any library just use:

- To Import a single function from a library
```python3
import libs.{your_library_name}.{function_name}
```
- To Import full library just use:
```python3
import libs.{you_library_name}
```
